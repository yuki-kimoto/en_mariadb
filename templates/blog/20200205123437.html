Remove foreign key constraint To remove the foreign key constraint, run the "alter table drop foreign key" command. 
alter table table name drop foreign key foreign key constraint name


This is a sample to remove the foreign key constraint called book_ibfk_1 in the book table.


alter table book drop foreign key book_ibfk_1
