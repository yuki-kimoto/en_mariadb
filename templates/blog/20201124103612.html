You can update the line using the update statement. 
#update statement-line update 
update table name set field name 1 = value 1, field name 2 = value 2, ... where condition;
When updating a row, it is basic to set a condition with where. If there is no where clause, all rows will be updated (wow, database recovery). This is a sample for updating a row. 
update book set name ='Perl', price = 3000 where id = 1;
