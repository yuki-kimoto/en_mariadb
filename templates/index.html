MariaDB入門

MariaDBは、MySQLからフォークされたMySQL互換性が高いリレーショナルデータベースシステムです。MySQLがOracleに買収されたときに、プロジェクトがフォークされました。 2020年現在、MariaDBとMySQLは、内部的なソースコードにかなりの差がでてきています。バージョン番号の付け方も変更がありました。外部的な機能の差が広がってくることも予想されます。 そこで、MariaDB用の解説サイトを作成することにしました。 新着情報

  • MariaDBデータベースに接続する

ユーザー管理

  • ユーザーを作成する - CREATE USER
  • パスワードの設定,データベースとテーブル権限の設定 - GRANTコマンド

データベースの管理

  • USE - データベースを選択する
  • データベースを作成する

テーブルの管理

  • CREATE TABLE - テーブルを作成する
  • テーブルの一覧を表示する

ディスク容量調査

  • MariaDBでテーブルの容量とデータベースの容量を確認する

データベースエンジン

  • データベースエンジンの種類を確認する
  • データベースエンジンを変更する

外部キー制約

  • 外部キー制約を確認する
  • 外部キー制約を削除する

インデックス

  • インデックスを作成する
  • インデックスを削除する
  • インデックスを確認する

性能調査

  • EXPLAIN - SQLの性能調査のためにSQLの実行プランを見る
