MariaDB入門

MariaDBは、MySQLからフォークされたMySQL互換性が高いリレーショナルデータベースシステムです。MySQLがOracleに買収されたときに、プロジェクトがフォークされました。 2020年現在、MariaDBとMySQLは、内部的なソースコードにかなりの差がでてきています。バージョン番号の付け方も変更がありました。外部的な機能の差が広がってくることも予想されます。 そこで、MariaDB用の解説サイトを作成することにしました。 新着情報

MariaDBデータベースへの接続

  • MariaDBデータベースに接続する

ユーザー管理

  • rootのパスワードを設定する
  • rootのパスワードを再設定する
  • ユーザーを作成する - create user
  • パスワードの設定,データベースとテーブル権限の設定 - grant文
  • ユーザーの一覧を表示する
  • ユーザーを削除する

データベースの管理

  • use - データベースを選択する
  • データベースを作成する
  • データベースの一覧を表示する - show databases文
  • データベースを削除する - drop database文

テーブルの管理

  • create table - テーブルを作成する
  • テーブルの一覧を表示する
  • テーブル名を変更する - rename table文

フィールド定義

  • フィールドの定義を変更する - alter table change column文
  • 日付時刻を表すdatetime型の現在時刻での初期化と更新

ディスク容量調査

  • MariaDBでテーブルの容量とデータベースの容量を確認する

データベースエンジン

  • データベースエンジンの種類を確認する
  • データベースエンジンを変更する

外部キー制約

  • 外部キー制約を確認する
  • 外部キー制約を削除する

インデックス

  • インデックスを作成する
  • インデックスを削除する
  • インデックスを確認する

性能調査

  • explain - SQLの性能調査のためにSQLの実行プランを見る

MariaDBの設定

  • MariaDBのデフォルトポートは3306
  • SQLモード「STRICT_TRANS_TABLES」を無効にする
